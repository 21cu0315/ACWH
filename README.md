# ACWH

***コントローラー必須、途中でコントローラー認識できない場合はエラーになる***
***XInputとDirectInputのコントローラーも対応***

Ps4のコントローラーを基準として、
□:メイン攻撃(長押し)
△:SPスキル
X:ジャンプ(二段ジャンプ可能)
L1:ダッシュ(空中ダッシュ可能)
R1:サブ攻撃(長押し)
TAB(キーボード):ゲームリセット

実行手順

実行ファイル\build\ACWH.exe を開いたら、遊べるようになります。



担当箇所：
・敵の処理と考案
・サウンド作成（ＢＧＭ作成、サウンドエフェクト）
・グラフィックエフェクト作成
（プレイヤー攻撃が敵に当たった時のエフェクトなど）
